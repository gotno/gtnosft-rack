<objective>
Replace `std::map` with `std::unordered_map` for the lights cache and eliminate the per-tick sort + `std::set_intersection` by using `std::unordered_set` membership tests.

Purpose: Reduce module lookup from O(log n) to O(1) and subscription filtering from O(n log n) to O(n).
Output: Modified `ModuleLightsBundler.hpp` and `ModuleLightsBundler.cpp` with unordered containers.
</objective>
